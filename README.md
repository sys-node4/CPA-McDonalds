# 🍟 McDonald's Rewards Landing Page

A high-performance, conversion-focused landing page designed for a McDonald's CPA (Cost Per Action) promotional offer. This project showcases a modern marketing UI built for speed and maximum user engagement.

## 🚀 Project Overview

In the world of CPA marketing, speed is everything. This landing page is built using a **single-file architecture**, housing all HTML, CSS, and JavaScript in one snippet. This eliminates external requests, ensuring the page loads almost instantly on mobile and desktop devices—a key factor in reducing bounce rates.

## ✨ Key Features

* **Conversion-Optimized UI:** Designed with psychological triggers, high-contrast CTAs, and trust signals.
* **Single-File Build:** HTML, CSS, and JS are bundled in one file for easy deployment and maximum portability.
* **Fully Responsive:** Mobile-first design that adapts perfectly to any screen size.
* **Modern Aesthetics:** Clean, vibrant McDonald's-themed design using semantic HTML5 and modern CSS3.
* **Lightweight:** Minimal code footprint for 100/100 performance scores.

## 🛠️ Tech Stack

* **HTML5** - Structured for clarity and accessibility.
* **CSS3** - Custom styling with a focus on brand-aligned colors and animations.
* **Vanilla JavaScript** - Logic for interactive elements and offer triggers.

## 📂 Deployment

Since this is a standalone `index.html` file, you can get it live in seconds:

1.  **Clone the Repo:**
    ```bash
    git clone [https://github.com/deicyde/CPA-McDonalds.git](https://github.com/deicyde/CPA-McDonalds.git)
    ```
2.  **Run Locally:** Just double-click `index.html` to view it in your browser.
3.  **Host for Free:** Drag and drop the file into [Netlify Drop](https://app.netlify.com/drop) or enable **GitHub Pages** in the repository settings.
